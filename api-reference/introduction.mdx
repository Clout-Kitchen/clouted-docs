## Base URLs

- Production: `https://app.clouted.com/api/v1`
- Staging: `https://staging.app.clouted.com/api/v1`

## Authentication

All endpoints are authenticated with a Bearer token. Include it on every request:

```http
Authorization: Bearer <YOUR_TOKEN>
```

## Resources

- **Campaigns** — manage clipping campaigns and import clip submissions.
- **Creators** — look up creator profiles and performance stats.
- **Tracking Campaigns** — manage tracking campaigns and import content URLs.
